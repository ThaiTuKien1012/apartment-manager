# 📘 Xây dựng Backend AI Image Retrieval (Node.js + MongoDB)

## 🎯 Mục tiêu

Xây dựng hệ thống backend cho phép:

* Upload và lưu trữ hình ảnh
* Gán metadata (mô tả, tags)
* Tìm kiếm hình ảnh bằng câu hỏi tự nhiên (AI hỗ trợ)
* Trả về ảnh phù hợp cho frontend

---

## 🏗️ Kiến trúc tổng thể

```
Client (FE)
   ↓
Express API (Node.js)
   ↓
MongoDB (metadata)
   ↓
Storage (S3 / Local)
   ↓
AI Service (embedding / NLP)
```

const imageSchema = new mongoose.Schema({
  url: String,
  description: String,
  tags: [String],
  embedding: [Number], // dùng cho AI search

## 🎮 5. Controller

### controllers/imageController.js

#### Upload ảnh

```javascript
import Image from "../models/Image.js";

export const uploadImage = async (req, res) => {
  try {
    const { description, tags } = req.body;

    const image = new Image({
      url: req.file.path,
      description,
      tags: tags.split(","),
    });

    await image.save();

    res.json(image);
  } catch (err) {
    res.status(500).json({ error: err.message });
  }
};
```

---

#### Search ảnh cơ bản (theo text)

```javascript
export const searchImages = async (req, res) => {
  try {
    const { query } = req.query;

    const images = await Image.find({
      $or: [
        { description: { $regex: query, $options: "i" } },
        { tags: { $in: [query] } },
      ],
    });

    res.json(images);
  } catch (err) {
    res.status(500).json({ error: err.message });
  }
};
```

## 🧠 6. AI Search (Nâng cao)

### services/aiService.js

```javascript
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});

export const getEmbedding = async (text) => {
  const res = await client.embeddings.create({
    model: "text-embedding-3-small",
    input: text,
  });

  return res.data[0].embedding;
};
```

---

### Lưu embedding khi upload

```javascript
import { getEmbedding } from "../services/aiService.js";

const embedding = await getEmbedding(description);

const image = new Image({
  url: req.file.path,
  description,
  tags: tags.split(","),
  embedding,
});
```

---

### Search bằng embedding

```javascript
import cosineSimilarity from "cosine-similarity";

export const searchByAI = async (req, res) => {
  const { query } = req.body;

  const queryEmbedding = await getEmbedding(query);

  const images = await Image.find();

  const results = images.map(img => ({
    ...img._doc,
    score: cosineSimilarity(queryEmbedding, img.embedding),
  }));

  results.sort((a, b) => b.score - a.score);

  res.json(results.slice(0, 5));
};
```

## 🛣️ 7. Routes

### routes/imageRoutes.js

```javascript
import express from "express";
import { upload } from "../utils/upload.js";
import {
  uploadImage,
  searchImages,
  searchByAI,
} from "../controllers/imageController.js";

const router = express.Router();

router.post("/upload", upload.single("image"), uploadImage);
router.get("/search", searchImages);
router.post("/search-ai", searchByAI);

export default router;
```

## 🚀 8. Khởi động server

### app.js

```javascript
import express from "express";
import dotenv from "dotenv";
import cors from "cors";
import { connectDB } from "./config/db.js";
import imageRoutes from "./routes/imageRoutes.js";

dotenv.config();
connectDB();

const app = express();

app.use(cors());
app.use(express.json());
app.use("/uploads", express.static("uploads"));

app.use("/api/images", imageRoutes);

app.listen(5000, () => {
  console.log("Server running on port 5000");
});
```
